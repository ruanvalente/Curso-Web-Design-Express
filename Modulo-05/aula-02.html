Em css precisamos entender um pequeno conceito de Classes e ID's. Cada um tem a sua respectiva função para a estilização dos elementos do documento, e assis também a sua especificação.

Podemos usar os ids para quando precisamos estilizar apenas um elemento do nosso documento, e não se pode repetir o mesmo ID na mesma página.

Quando precisamos estilizar mais de um elemento, usamos as classes css, pois usando as classes podemos estilizar mais de um elemento usando a mesma classe de estilização para o mesmo.

Podemos abrir a tag style dentro do nosso head para começar a fazer as nossas estiização.
