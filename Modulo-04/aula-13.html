Em html5 podemos usar a tag form para criar um Formulário para fazer nossos envios, então tudo que estiver entre a tag form será considerado parte do formulário.

Temos algumas tag dentro do form para ter melhor experiência do usuário.

A tag input, está tag espera que usuário entre com alguma informação porém ela temos que especificar o tipo de informação, ex:

  • type text: Seria o campo de texto referente ao formulário
  • type email: Seria o campo de texto onde poderiamos digitar o nosso e-mail dentro do nosso formulário
  • type submit: Seria o nosso botão dentro do formulário e precisamos passar um valor ( value ) que seria o nome do nosso botão.
  • Text area: Seria uma area de texto, onde podemos digitar informações
  • Checkbox: Seria checkgem em pequenos blocos
  • Radio: Seria pequenos circulos para fazer marcaçoes de opicoes
  • Select: Onde passamos option e dentro de option mostramos as opções aos usuários
  • Option: Um tag onde usamos para realizar opçoes
  • Label: Mostramos a label do site.
  • Button: Mostra um botão mas sem válidação de campo.

Dentro de text e email podemos passar uma outra tag que é a required que faz a válidação do que foi digitado e também temos a tag placeholder
