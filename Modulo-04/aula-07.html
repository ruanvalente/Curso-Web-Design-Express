Basicamente, sections são sessões do nosso site. Assim podemos ter varios tipos de sessões no site, porém cada uma referenciando a sua respectiva sessão.

As sections se comportam como as divs, porém em um contexto mais semantico as sections são melhor aprovieitadas do que as divs.

Ex:

Está sessão é para o inicio do meu site

Está outra sessão é para o corpo do meu site

Está sessão é para meu roda-pé do meu site

OBS: Não usamos sections para demarcar o nosso roda-pé, para isso usamos uma tag especifica do html5, o footer.
